Full Stack Website (E-Commerce)

I designed and developed a music e-commerce website from the ground up. For the frontend, I utilized HTML, CSS, and JavaScript, along with my own NPM module, to structure and style the product listings and prices. The majority of user interactions were implemented using Vanilla JS, while customer payments were securely processed via the Stripe API and associated webhooks.

On the backend, I created a custom server script to automate email receipt generation after each purchase. Additionally, I leveraged Express and Axios to facilitate product filtering and search functionalities, with product tags stored in metadata for efficient categorization.

To ensure a cohesive and reusable design, I used Express and EJS to develop visual components for the shopping page, enabling consistent and flexible product displays across the site.
